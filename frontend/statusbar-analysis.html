MOVE TO TOAST Temporary Notifications

Notification Messages
Success/error/warning/info messages from user actions (save settings, etc.)
Toast → Auto-dismiss 5s
Scan Completion
"Scan complete: 5 new videos found" - temporary result message
Toast → Auto-dismiss 5s
Error Messages
Download failures, API errors - currently auto-fades after 10s
Toast → Auto-dismiss 10s
Cookie Warnings
Authentication issues with YouTube cookies
Toast → Click to dismiss
Active Scan Status
"Scanning TechChannel..." with spinner - in-progress operation
Toast → Persists until done
Queue Paused
"Queue paused" warning when downloads are held
Toast → Persists until resumed
Delay Info
"Waiting 30s before next download..." between downloads
Toast → Auto-dismiss on start

REMOVE / CONSOLIDATE Redundant or Unnecessary

Queue Count
"[5 queued]" - already shown as badge on Queue tab in navigation
Remove → Use nav badge
Auto-refresh Status
"Auto-refresh enabled for 03:00" - informational, shown in Settings
Remove → Shown in Settings
Idle State
"Idle" - no value, absence of activity is self-evident
Remove → No replacement needed

NEEDS ALTERNATIVE Persistent or Special Items

Download Progress
Real-time: speed (2.4 MB/s), ETA (3:45), progress (67%) - needs visibility during downloads
Sticky Toast → Updates live
Log Button
Expandable quick logs panel - useful for debugging but not essential
Move → Settings page only

Final Recommendation

  • Remove the status bar entirely - all useful info moves to toast notifications
  • Toast stack in bottom-right corner - multiple toasts can stack vertically
  • Download progress toast is "sticky" - stays visible during active downloads, dismisses on completion
  • Scan/operation toasts persist until operation completes, then show result
  • User action toasts (save, delete, etc.) auto-dismiss after 5 seconds
  • Error toasts require manual dismissal or auto-dismiss after 10 seconds
  • Log viewer moves to Settings page - rarely needed, doesn't need persistent access
  • Queue count is already visible in the nav tab badge - no duplication needed
